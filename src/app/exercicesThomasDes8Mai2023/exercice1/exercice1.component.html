Exercice 01 - Créer son profil

Vous devez créer une application Angular pour permettre aux utilisateurs de créer leur propre profil personnel.
L'application doit contenir les champs suivants.

  • Nom complet (string) : {{ nomComplet }}
  • Adresse e-mail (string) : {{ adresseMail }}
  • Date de naissance (date) : {{ dateDeNaissance }}
  • Genre (string) : {{ genre }}
  • Langues parlées (string) : {{ languesParlees }}
  • Image de profil (optionnelle) (any[]) : {{ photoDeProfil }}
